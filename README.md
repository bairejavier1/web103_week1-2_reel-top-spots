# WEB103 Project 2 - *Reel-Top-Spots*

Submitted by: **Baire Diaz**

About this web app: **A dynamic web app that displays top fishing spots in South Florida, fetching data from a PostgreSQL database and showing detailed information for each spot. The app includes search functionality and a back button for improved navigation.**

Time spent: **4** hours
